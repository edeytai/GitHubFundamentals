Preguntas y Respuestas

  1. ¿Qué es el DOM?

    El DOM (Document Object Model) es una representación estructurada de un documento HTML o XML, en forma de árbol, que permite acceder y manipular el contenido, la estructura y los estilos. Referencia MDN

  2. ¿Qué es un nodo?

    Un nodo es la unidad básica del DOM. Puede representar un elemento, un texto, un atributo, etc., y forma parte de la estructura jerárquica del documento. Referencia MDN

  3. ¿Cómo se accede a un elemento del DOM?

    Se puede acceder a los elementos del DOM mediante métodos como getElementById, getElementsByClassName, getElementsByTagName, querySelector o querySelectorAll. Referencia MDN

  4. ¿Cómo se modifica un elemento del DOM?

    Un elemento del DOM se puede modificar cambiando sus propiedades (por ejemplo, textContent o innerHTML) o usando métodos como setAttribute para alterar sus atributos. Referencia MDN

  5. ¿Cuál es la diferencia entre getElementby* y querySelector*?

    Los métodos getElementBy* (por ejemplo, getElementById o getElementsByClassName) son específicos y retornan uno o varios elementos según el criterio; en cambio, querySelector y querySelectorAll utilizan selectores CSS, lo que permite búsquedas más flexibles (retornando el primer elemento o todos los elementos que coincidan). Referencia MDN

  6. ¿Qué es un evento?

    Un evento es una acción o suceso que ocurre en el sistema (como un clic, movimiento del mouse o carga de página) y que puede ser detectado y gestionado mediante JavaScript. Referencia MDN

  7. ¿Qué es un listener?

    Un listener es una función que espera a que ocurra un evento específico y ejecuta una acción cuando dicho evento se dispara. Referencia MDN

  8. ¿Qué es un callback?

    Un callback es una función que se pasa como argumento a otra función y se ejecuta cuando ocurre un evento o al completarse una tarea. Referencia MDN

  9. ¿Qué es un objeto de evento?

    Un objeto de evento es una instancia que contiene información sobre el evento que ocurrió, como el tipo de evento, el elemento que lo disparó y otros detalles relevantes. Referencia MDN

¿Cuándo y para qué usarías ChatGPT y cuándo Copilot?

Utilizaria ChatGPT cuando necesite soluciones a profundidad o darle prompt mas a detalle, Copilot lo utilizaria para algo mas personal o una duda mas simple.
