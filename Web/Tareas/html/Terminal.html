Terminal

En esta pagina de Terminal se dará un repaso a todos los comandos que nos permiten navegar y utilizar la terminal.

¿Qué es la Terminal?

La terminal es una interfaz de línea de comandos que permite a los usuarios comunicarse con el sistema operativo mediante texto. A través de ella, se pueden ejecutar programas, navegar por directorios, manipular archivos y realizar configuraciones avanzadas sin necesidad de una interfaz gráfica.

A continuación se presenta una tabla con los comandos mas relevantes:

Comando Descripción
ls Muestra el contenido del directorio actual.
cd [directorio] Cambia al directorio especificado.
pwd Muestra la ruta del directorio actual.
cd .. Cambia al directorio anterior al actual.
mkdir [nombre] Crea un nuevo directorio.
rm [archivo] Elimina un archivo.
rm -r [directorio] Borra un directorio y su contenido.
cp [origen] [destino] Copia un archivo o directorio al destino especificado.
mv [origen] [destino] Mueve un archivo o directorio al destino especificado.
touch [archivo] Crea un archivo vacío.
cat [archivo] Muestra el contenido de un archivo.
man [comando] Muestra el manual de un comando.
history Muestra el historial de comandos utilizados
vim [archivo] Edita un archivo desde la terminal.

i: Permite editar el archivo.

Tecla esc: Sale del modo de edición.

:w : Guarda los cambios del archivo.

:q : Salir del editor.

:!q : Confirma la salida del editor sin guardar cambios.

:wq : Guarda el archivo y sale del editor.

clear Limpia la pantalla de la terminal.
code . Abre VSCode en la dirección actual.
head/tail [archivo] -n Visualiza las primeras o ultimas lineas de un archivo.
