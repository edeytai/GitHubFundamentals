CSS

En esta sección hablaremos sobre qué es CSS, selectores, propiedades y valores

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que define el estilo y diseño de páginas web, controlando aspectos como colores, fuentes, márgenes y disposición de elementos.

¿Qué son los selectores?

Los selectores de CSS son patrones que permiten aplicar estilos a elementos HTML específicos. Pueden ser básicos como etiquetas (p, div), clases (.clase), IDs (#id), o avanzados como combinadores (div > p), pseudo-clases (:hover) y pseudo-elementos (::before).

Propiedades

CSS tiene muchas propiedades, pero las principales se agrupan en categorías:

  • Texto: color, font-size, font-family, text-align, line-height.
  • Fondo: background-color, background-image, background-size.
  • Caja (Box Model): width, height, margin, padding, border.
  • Posicionamiento: position, top, left, right, bottom, z-index.
  • Visualización y Flexibilidad: display, visibility, flex, grid.
  • Animaciones: transition, animation, transform.
Valores

Los valores en CSS dependen de cada propiedad, pero estos son los más comunes:

Propiedad Valores
Colores nombres (red), hexadecimales (#ff0000), RGB (rgb(255, 0, 0)).
Tamaños píxeles (px), porcentajes (%), em/rem (1em, 2rem), vh/vw.
Posicionamiento static, relative, absolute, fixed, sticky.
Display block, inline, flex, grid, none.
Bordes solid, dashed, dotted, double, none.
Textos bold, italic, underline, uppercase, lowercase.
Flex/Grid flex-start, center, space-between, grid-template-columns.
Animaciones ease, linear, infinite, rotate(45deg).
